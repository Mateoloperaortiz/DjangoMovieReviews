About

Welcome to a Simple Django App for Movies Reviews, your go-to place for movie reviews!

This application is a simple web app built with Python and Django for Proyecto Integrador 1. It allows users to browse movies, search for movies by title, and in the future, it will allow users to post reviews.

We are passionate about movies and wanted to create a platform to share our thoughts and opinions with the world.

This project is part of a workshop to learn about web development with Django, Git, and GitHub.
